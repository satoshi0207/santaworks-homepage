### デザイン方針（確定）

- **配色**: 白／オフホワイト基調 × サンタレッドのアクセント（ボタン・見出し・装飾）
- **ヒーロー**: 世界観フル演出。背景全体を**「光の粒＝記憶の欠片」がゆっくり舞い降りる**没入型。
  雪と断定しない抽象表現にする（真夏でも成立する通年モチーフ）。粒は柔らかく明滅し、
  少数（15%程度）だけアクセントの赤。ライト=明るい静けさ、ダーク=夜の静けさの二面
- **デザインモック**: 方向性確認用のカンプを Claude Artifact で作成済み
  https://claude.ai/code/artifact/e002b0c2-6494-4ce0-86dd-0465ee9858de
  （単一HTML+CSS+Vanilla JS。本実装のビジュアル設計図であり、コードは流用前提ではない）
- **フォント**: ヒラギノ角ゴ / Noto Sans JP 系のスタンダードゴシック
- **アニメーション**: モダンでさりげなく。過剰な演出はしない（ヒーローの粒以外は控えめ）。
  出現演出は「ふわぁ」系で統一 — ぼかし(blur)が晴れながら浮かび上がる長め（1〜1.6s）のフェード。

- **ヒーローコピーは1行**: PCでは1行表示、狭い画面では読点の位置でのみ折り返す（フレーズ単位のinline-block）
- **モバイルナビはハンバーガーメニュー**（全画面オーバーレイ型）。下部固定CTAの追加は提案中・未決定
- **問い合わせ**: 無料フォームサービス（Formspree等）を埋め込み

## 実装メモ

- Next.js 16 (App Router / TS) + Tailwind CSS v4 + framer-motion
- デザイントークンは `app/globals.css` の CSS変数 + `@theme inline`（ライト/ダークは prefers-color-scheme 自動）
- 「ふわぁ」出現は `components/motion.tsx` の FadeIn（読み込み時）/ Reveal（スクロール時）
- 光の粒は `components/Motes.tsx`（Canvas、reduced-motion対応）
- 問い合わせフォームは Formspree。フォームID `xkodjjgn` を `components/ContactForm.tsx` に
  直書き済み（IDは公開情報なので問題ない）。環境変数 `NEXT_PUBLIC_FORMSPREE_ID` で上書き可能
- デプロイはCI/CD化済み: mainにpushするとCloudflareが自動でビルド＆デプロイ（1〜2分）
- Google Analytics 4: `app/layout.tsx` に実装済み。**環境変数 `NEXT_PUBLIC_GA_ID`（例 G-XXXXXXXXXX）が
  セットされたビルドでのみ有効化**（未設定なら無効）。Cloudflareのビルド環境変数に設定→再デプロイで計測開始。
  プライバシーポリシー（/privacy）にGA利用・Cookie・オプトアウトを明記済み
- テーマ: `data-theme` 属性方式（既定ダーク）。`components/ThemeToggle.tsx` で手動切替、localStorage保存。
  ダーク判定はTailwindの `@custom-variant dark` を data-theme に紐付け（`app/globals.css`）
- 規約ページ: プライバシーポリシー `/privacy`、特定商取引法に基づく表記 `/legal`。フッターから導線。
  特商法は住所・電話を「請求があれば遅滞なく開示」の例外運用（消費者庁ガイド準拠）
